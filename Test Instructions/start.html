Test Session

Set up the EHi

1) Double-click the Test-master.zip file. Open the Test-master folder, then drag-and-drop the Test_Files folder to your raptor's home directory.


2) Log onto raptor by opening PuTTY, then enter "raptor.kent.ac.uk" in the Host Name field.


2) When prompted to, enter your kent username (eg. abc1) and password.


3) In PuTTY, change to the Test-master directory by entering the command "cd Test_Files"


4) Change the permission rights of the ehi.bat file by executing the command "chmod u+x ehi.bat".


5) Execute the command "./ehi.bat" .


6) The EHi program should now start.
