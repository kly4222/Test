Test Session

Tasks

1) Create a factorial function.
This will take a non-negative integer n and calculate the product of n * (n-1) * (n-2) * ... * 2 * 1. Note that the factorial of 0 is 1, and calculating the factorial of any negative number should return -1.
fact :: Integer -> Integer

2) Create a function that will compare the first element of 2 pairs.
compareFst :: (a,b) -> (a,b) -> Bool

3) Create a function that will take a list of tuples and swap their contents.
swapTuple :: [(a,b)] -> [(b,a)]

4) Create a function that will sort a list of Strings alphabetically, removing duplicates.
aToZ :: [String] -> [String]

5) Create a function that will return the middle element of a list (the middle two elements if the length of the list is even, and none if the list is empty).
getMiddle :: [a] -> [a]
