Set up the EHi

1) Double-click the Test-master.zip file, then drag-and-drop the Test-master folder to your raptor's home directory.


2) Log onto raptor by opening PuTTY, then enter "raptor.kent.ac.uk" in the Host Name field.


2) When prompted to, enter your kent username (eg. abc1) and password.


3) In PuTTY, change to the Test-master directory by entering the command "cd Test-master"


4) Execute the command "./ehi.bat"


5) The EHi program should now start.
